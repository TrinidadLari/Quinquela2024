FIELTRO

Profesora Mónica Tassi

ADULTOS MAYORES DE 16 AÑOS

Taller de Fieltro (tejido ancestral). Único tejido que no se teje... se amasa!

- Conocerán la materia prima, el vellón de lana. - Aprenderán las técnicas básicas del fieltro. - Harán la tela de fieltro, lisa y con intervenciones. - Modelaran el vellón para hacer bolitas, rastas, flores. - Agujaran el vellón para realizar bordados, muñequitos. Para la primera clase no traer material. Será una clase explicativa.
